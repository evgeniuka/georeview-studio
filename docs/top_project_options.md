# Top Project Options

## 1. GeoReview Studio: Universal OSM/GIS Review Workbench

Recommended portfolio direction.

Why it works:

- shows GIS data engineering;
- supports multiple analysis profiles;
- has API, UI, validation and reports;
- can grow beyond one city;
- keeps claims careful and evidence-based.

## 2. Safe Access Israel

Best first domain story.

Why it works:

- clear public-interest topic;
- easy to explain to non-GIS reviewers;
- current data supports a Kfar Saba pilot;
- outputs field-review candidates, not crash predictions.

## 3. Open Mobility Evidence Lab

Best expansion theme.

Why it works:

- includes transit stops, cycling, parks and pedestrian infrastructure;
- can reuse the same source onboarding and profile registry;
- creates a broader civic-tech portfolio narrative.

## Recommendation

Build option 1 as the application architecture and keep option 2 as the polished first profile.
