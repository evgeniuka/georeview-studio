# Repository Dry-Run Review

v077 adds a local dry-run review for the final public repository archive.

It does not publish a repository. It creates a small ZIP with:

- archive structure preview;
- final public sharing checklist;
- evidence references;
- claim boundaries;
- hard-exclude rules for source GIS files.

The dry-run keeps source GIS files local and treats OSM-derived outputs as infrastructure review evidence only.
