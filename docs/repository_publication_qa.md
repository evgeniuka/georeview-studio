# Repository Publication QA

v077 keeps the repository-publication quality gate after the GitHub-ready publication bundle.

The module verifies:

- README, portfolio case study and repository file manifest exist.
- The approved field-review wording is present.
- Source GIS data is not included in the publication ZIP.
- Validation and API contract evidence are current for the release.
- Public sharing has a clear walkthrough and reviewer attention items.

This is still a local-first review artifact. It does not publish externally and does not mutate source GIS files.
