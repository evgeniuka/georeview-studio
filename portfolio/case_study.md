# GeoReview Studio Portfolio Case Study

## Project

GeoReview Studio is a local-first GIS analytics workbench for infrastructure review prioritization.

## Pilot

Safe Access Kfar Saba, using OpenStreetMap / Geofabrik source data.

## Result

- Pedestrian generators: 391
- Crossings: 342
- Road segments: 2603
- Traffic-calming features: 11
- Raw PBF enrichment: True
- Route-aware rows: 391
- Median route distance to mapped crossing: 218.6 m
- Generators over 250 m by route proxy: 169

## Correct Interpretation

This location has infrastructure risk indicators and should be reviewed on-site.

Missing OSM tags are data-quality flags, not proof that infrastructure is absent.
